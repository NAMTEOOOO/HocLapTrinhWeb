Công dụng các thẻ trong html

 Thẻ <code></code> có ý nghĩa giúp chèn vào đoạn văn bản hiển thị lên các đoạn code
 Thẻ từ <h1></h1> đến <h6></h6> là để biểu diễn tiêu đề từ lớn tới nhỏ
 Thẻ <pre></pre> giúp giữ nguyên format của văn bản ví dụ như các đoạn dấu cách
 Thẻ <p></p> Là thẻ chèn văn bản 
 Thẻ <br/> Là thẻ xuống dòng có thể thêm vào sau đít 1 văn bản
 Thẻ <hr/> Là thẻ Chèn thêm 1 đường  ngang phân cách
 Thẻ <strong>Chữ đậm</strong> Là thẻ chữ đậm (quan trọng)
 Thẻ <b></b> Là thẻ chữ đậm (không quan trọng)
 Thẻ <em></em> Là thẻ chữ nghiêng (quan trọng)
 Thẻ <i></i> Là thẻ chữ nghiêng (không quan trọng)
 Thẻ <u></u> Là thẻ chữ gạch chân
 Thẻ <mark></mark> Là thẻ chữ hightlight

Các thẻ khác

 Thẻ <a></a> dùng để tạo liên kết đến trang khác hoặc phần tử khác trên trang
 Thẻ <img> dùng để hiển thị hình ảnh
 Thẻ <ul></ul> tạo danh sách không thứ tự (bullet)
 Thẻ <ol></ol> tạo danh sách có thứ tự (số thứ tự)
 Thẻ <li></li> là từng phần tử trong danh sách

 Thẻ <div></div> tạo khối chứa nội dung (block element)
 Thẻ <span></span> tạo vùng chứa inline nhỏ
 Thẻ <table></table> tạo bảng
 Thẻ <tr></tr> là một dòng trong bảng
 Thẻ <th></th> là ô tiêu đề
 Thẻ <td></td> là ô dữ liệu

 Thẻ <form></form> tạo biểu mẫu nhập liệu
 Thẻ <input> tạo ô nhập dữ liệu (text, checkbox, radio...)
 Thẻ <textarea></textarea> tạo ô nhập nhiều dòng
 Thẻ <button></button> tạo nút nhấn
 Thẻ <select></select> tạo danh sách thả xuống
 Thẻ <option></option> là từng mục trong select

 Thẻ <header>, <footer>, <section>, <nav> được dùng để phân chia các phần chính của trang web theo dạng bố cục HTML5 hiện đại

Tips khi viết code HTML

✅ 1. **Luôn bắt đầu với doctype**
   Thêm <!DOCTYPE html> ở đầu file để trình duyệt hiểu rằng bạn dùng HTML5

✅ 2. **Sử dụng cấu trúc cơ bản đầy đủ**
   Mỗi trang HTML nên có đủ:
   
   <html>, <head>, <meta>, <title>, <body>
   

✅ 3. **Sử dụng thụt dòng (indentation)**
   Giúp mã rõ ràng, dễ đọc, dễ bảo trì
   Ví dụ:
   
   <ul>
     <li>Mục 1</li>
     <li>Mục 2</li>
   </ul>
   

✅ 4. **Thẻ tự đóng phải có dấu "/"**
   Ví dụ: <br />, <hr />, <img />

✅ 5. **Luôn khai báo thuộc tính "alt" cho ảnh**
   Giúp cải thiện SEO và hỗ trợ người khiếm thị:
   <img src="img.png" alt="Ảnh mô tả nội dung" />

✅ 6. **Dùng thẻ semantic thay vì chỉ dùng <div>**
   Giúp SEO tốt hơn và dễ hiểu hơn:
   <header>, <main>, <footer>, <article>

✅ 7. **Đặt đường dẫn đúng**
   - ./ dùng để chỉ thư mục hiện tại
   - ../ dùng để đi lên 1 cấp thư mục

✅ 8. **Kiểm tra file bằng trình duyệt**
   Dùng Chrome hoặc Firefox mở file `.html` để xem kết quả trực tiếp

✅ 9. **Sử dụng Comment để ghi chú**
   Ghi chú không hiển thị trên trình duyệt:
   <!-- Đây là phần giới thiệu -->

✅ 10. **Kiểm tra lỗi với DevTools**
   Dùng F12 trên trình duyệt để kiểm tra HTML, CSS và Console lỗi

✅ 11. **Học thêm bằng tài liệu chính thức**
   Truy cập: MDN Web Docs để xem hướng dẫn HTML chuẩn nhất

Hướng dẫn chèn ảnh trong HTML

🔹 Cú pháp cơ bản:
  <img src="duong-dan-anh" alt="mo ta anh" />

-------------------------
📁 Có 2 loại đường dẫn:

1️⃣ Đường dẫn tuyệt đối (absolute path):
- Dẫn từ trang web hoặc máy chủ ngoài
  <img src="https://example.com/images/avatar.jpg" alt="Ảnh từ web ngoài" />

👉 Ưu điểm: Có thể dùng ở mọi nơi  
👉 Nhược điểm: Phụ thuộc vào kết nối mạng

2️⃣ Đường dẫn tương đối (relative path):
- Dựa trên vị trí file HTML hiện tại

📌 Ví dụ:
  <img src="./Img/avatar.png" alt="Ảnh cùng cấp" />
  <img src="../assets/logo.png" alt="Ảnh từ thư mục cha" />

📌 Quy tắc định hướng đường dẫn:
  🔸 ./       → thư mục hiện tại  
  🔸 ../      → đi lên 1 cấp thư mục  
  🔸 ../../   → đi lên 2 cấp  
  🔸 folder/  → vào thư mục con  

-------------------------
💡 Mẹo khi dùng ảnh:

✅ Luôn viết thuộc tính alt cho ảnh
✅ Không dùng dấu cách trong tên file ảnh  
✅ Nên đặt ảnh trong thư mục riêng như /Img  
✅ Ưu tiên dùng ảnh định dạng .jpg, .png, .webp
✅ Kiểm tra đường dẫn cẩn thận nếu ảnh không hiện

-------------------------
📎 Chèn ảnh làm liên kết:
- Gói thẻ <img> trong thẻ <a>

📌 Ví dụ:
  <a href="profile.html">
    <img src="./Img/avatar.png" alt="Ảnh đại diện" />
  </a>

Hướng dẫn chèn liên kết (link) trong HTML

🔹 Cú pháp cơ bản:
  <a href="duong-dan" target="_blank">Nội dung hiển thị</a>

-------------------------
📁 Có 2 loại đường dẫn:

1️⃣ Đường dẫn tuyệt đối (absolute link):
- Dẫn đến một trang web bên ngoài
  <a href="https://www.google.com" target="_blank">Google</a>

👉 Dùng khi liên kết tới website ngoài  
👉 target="_blank": Mở liên kết trong tab mới

2️⃣ Đường dẫn tương đối (relative link):
- Dẫn đến trang trong cùng project

📌 Ví dụ:
  <a href="./MoreInfo/contact.html">Liên hệ</a>
  <a href="../index.html">Trang chủ</a>

📌 Quy tắc định hướng đường dẫn:
  🔸 ./       → thư mục hiện tại  
  🔸 ../      → đi lên 1 cấp thư mục  
  🔸 folder/  → vào thư mục con  

-------------------------
💡 Mẹo khi dùng liên kết:

✅ Dùng target="_blank" cho link ngoài để không rời trang chính  
✅ Ghi rõ ràng nội dung link (tránh "bấm vào đây")  
✅ Kiểm tra kỹ đường dẫn nếu không hoạt động  
✅ Dùng comment để chú thích:
   <!-- Link tới trang kỹ năng cá nhân -->

✅ Dùng đường dẫn tương đối để dễ di chuyển project giữa các máy

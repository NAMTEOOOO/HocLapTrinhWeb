Css Training

Có 3 nơi có thể đặt CSS
  - Inline CSS nội tuyến <tag style="css"></tag>
  - Internal CSS nội bộ <style>css</style>
  - External CSS bên ngoài <link rel="stylesheet" href="style.css">
  - Inline là ghi vào bên trong thẻ bằng style=" "
  - Internal là ghi vào cặp thẻ style bên trên thẻ body
  - External là ghi file bên ngoài
